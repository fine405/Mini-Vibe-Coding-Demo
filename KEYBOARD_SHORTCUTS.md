# Keyboard Shortcuts

## Command Palette
- **⌘K** - Open command palette
- **⌘P** - Quick open file (also opens command palette)

## File Operations
- **⌘S** - Save workspace to browser storage

## Edit Operations
- **⌘⇧A** - Accept all changes (mark all files as clean)

## Layout
- **⌘1** - Toggle Chat Panel
- **⌘2** - Toggle Console Panel

## Chat
- **⌘Enter** or **Ctrl+Enter** - Send chat message (in chat input)

## Command Palette Features

The command palette provides quick access to:

### File Group
- **Save Workspace** - Save all files to IndexedDB
- **Open File** - Quick open file by fuzzy search

### Edit Group
- **Accept All Changes** - Accept all pending file modifications

### Layout Group
- **Toggle Chat Panel** - Show or hide the chat panel
- **Toggle Console Panel** - Show or hide the console panel

### File Search
- Type to search files by name with fuzzy matching
- Shows file status (New/Modified)
- Displays up to 10 matching files
- Automatically filters as you type

## Usage Tips

1. Press **⌘K** or **⌘P** to open the command palette
2. Start typing to search for files or commands
3. Use arrow keys to navigate results
4. Press **Enter** to select
5. Press **Escape** to close
